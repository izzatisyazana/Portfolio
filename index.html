    Project Al-Islah E-Derma Management System

    The Al-Islah e-Derma Management System is a web-based platform developed to streamline the management of donations (derma) received by mosques. This system, built using HTML, CSS, and JavaScript for the frontend, with a backend supported by MySQL and phpMyAdmin, allows mosque administrators to efficiently handle donor information and donation records.

    As a collaborative effort, I served as the front-end developer, ensuring a user-friendly interface for the system.     Project Haccers

    "Haccers" is a mobile application developed as a final year project by Izzati Syazana binti Abdul Rahman, utilizing Flutter and Firebase. Designed specifically for river trekking at the Sungai Chiling Fish Sanctuary, this application aims to assist hikers in navigating the trail, ensuring they do not lose their way. In addition to providing navigational support, the application includes features such as creating and editing user profiles for both hikers and administrators. It also offers detailed information about the Sungai Chiling Fish Sanctuary and a checklist of essential items for hikers. Administrators have the capability to add or edit information about the sanctuary and monitor the whereabouts of hikers as they check in at various trail checkpoints.
